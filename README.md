## 📖 Project Overview

This project is my **Week 1 Web Foundations Portfolio** created during the **IYF Weekend Academy Season 10** program.

The goal of the project was to practice:

- Writing **semantic HTML**
- Structuring a **multi-page website**
- Applying **basic accessibility principles**
- Publishing a website using **GitHub Pages**

The portfolio demonstrates my **early progress as a web development student** and showcases several mini-projects and challenges completed during the week.

## 📅 Daily Challenges Completed

| Day | Challenge |
|----|-----------|
| **Day 1** | Used 5 new HTML tags: `<abbr>`, `<blockquote>`, `<cite>`, `<code>`, `<mark>` |
| **Day 2** | Created a semantic navigation menu with a logo |
| **Day 3** | Added a contact section using `<address>` |
| **Day 4** | Built a photo gallery using `<figure>` and `<figcaption>` |
| **Day 5** | Designed a blog layout using `<article>`, `<time>`, and `<cite>` |
